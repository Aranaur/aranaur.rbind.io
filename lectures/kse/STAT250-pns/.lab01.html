Класична ймовірність та комбінаторика

Задача 1: Вибірка з урни

В урні містяться 5 білих і 3 чорних кулі. Навмання виймають 3 кулі. Знайти ймовірність того, що серед вийнятих куль буде рівно 2 білих і 1 чорна.

Математичне розв’язання

Загальна кількість способів вибрати 3 кулі з 8:

\[|\Omega| = \binom{8}{3} = \frac{8!}{3! \cdot 5!} = 56\]

Кількість сприятливих наслідків (2 білих з 5 та 1 чорна з 3):

\[|A| = \binom{5}{2} \cdot \binom{3}{1} = 10 \cdot 3 = 30\]

Ймовірність події \(A\):

\[P(A) = \frac{|A|}{|\Omega|} = \frac{30}{56} = \frac{15}{28} \approx 0{,}536\]

Розв’язання в R

# Кількість способів вибрати 2 білих з 5
white_ways <- choose(5, 2)

# Кількість способів вибрати 1 чорну з 3
black_ways <- choose(3, 1)

# Загальна кількість способів вибрати 3 кулі з 8
total_ways <- choose(8, 3)

# Ймовірність події
prob <- (white_ways * black_ways) / total_ways
cat(sprintf("P(A) = %d * %d / %d = %.4f", white_ways, black_ways, total_ways, prob))
P(A) = 10 * 3 / 56 = 0.5357

Задача 2: Правило додавання

Гральний кубик кидають один раз. Знайти ймовірність того, що випаде парне число або число, більше за 4.

Математичне розв’язання

Простір елементарних подій: \(\Omega = \{1, 2, 3, 4, 5, 6\}\), \(|\Omega| = 6\).

Визначимо події:

  • \(A = \{2, 4, 6\}\) — випаде парне число, \(P(A) = \frac{3}{6} = \frac{1}{2}\)
  • \(B = \{5, 6\}\) — випаде число більше 4, \(P(B) = \frac{2}{6} = \frac{1}{3}\)
  • \(A \cap B = \{6\}\) — обидві події, \(P(A \cap B) = \frac{1}{6}\)

За формулою включень-виключень:

\[P(A \cup B) = P(A) + P(B) - P(A \cap B) = \frac{1}{2} + \frac{1}{3} - \frac{1}{6} = \frac{3}{6} + \frac{2}{6} - \frac{1}{6} = \frac{4}{6} = \frac{2}{3}\]

Розв’язання в R

tibble(x = 1:6) |>
  mutate(
    A = x %% 2 == 0,
    B = x > 4
  ) |>
  summarise(
    `P(A)`   = mean(A),
    `P(B)`   = mean(B),
    `P(A∩B)` = mean(A & B),
    `P(A∪B)` = mean(A | B)
  )
# A tibble: 1 × 4
  `P(A)` `P(B)` `P(A∩B)` `P(A∪B)`

1    0.5  0.333    0.167    0.667

Умовна ймовірність та незалежність

Задача 3: Діти в сім’ї

У сім’ї двоє дітей. Відомо, що хоча б одна з них — дівчинка. Яка ймовірність того, що обидві діти — дівчинки?

Математичне розв’язання

Простір елементарних подій (Х — хлопчик, Д — дівчинка):

\[\Omega = \{(Х,Х),\; (Х,Д),\; (Д,Х),\; (Д,Д)\}\]

Визначимо події:

  • \(A = \{(Д,Д)\}\) — обидві дівчинки, \(P(A) = \frac{1}{4}\)
  • \(B = \{(Х,Д), (Д,Х), (Д,Д)\}\) — хоча б одна дівчинка, \(P(B) = \frac{3}{4}\)

Оскільки \(A \subset B\), маємо \(A \cap B = A\).

Умовна ймовірність:

\[P(A \mid B) = \frac{P(A \cap B)}{P(B)} = \frac{P(A)}{P(B)} = \frac{1/4}{3/4} = \frac{1}{3} \approx 0{,}333\]

Розв’язання в R

set.seed(123)
n_sim <- 100000

tibble(
  child1 = sample(c("Х", "Д"), n_sim, replace = TRUE),
  child2 = sample(c("Х", "Д"), n_sim, replace = TRUE)
) |>
  mutate(
    has_girl   = child1 == "Д" | child2 == "Д",
    both_girls = child1 == "Д" & child2 == "Д"
  ) |>
  filter(has_girl) |>
  summarise(
    `P(обидві | є хоча б одна)` = mean(both_girls),
    `Теоретичне значення`        = 1 / 3
  )
# A tibble: 1 × 2
  `P(обидві | є хоча б одна)` `Теоретичне значення`
                        <dbl>                 <dbl>
1                       0.333                 0.333

Задача 4: Незалежність подій

Кидають два гральних кубики. Визначити, чи є незалежними події:

  • \(A\) — на першому кубику випало парне число
  • \(B\) — сума очок більша за 7

Математичне розв’язання

\(|\Omega| = 36\) (усі пари \((i, j)\), де \(i, j \in \{1,\ldots,6\}\)).

Подія \(A\): перший кубик показує 2, 4 або 6:

\[P(A) = \frac{3 \cdot 6}{36} = \frac{18}{36} = \frac{1}{2}\]

Подія \(B\): сума \(> 7\) (можливі суми 8, 9, 10, 11, 12):

Сума Кількість
8 5
9 4
10 3
11 2
12 1

\[P(B) = \frac{5+4+3+2+1}{36} = \frac{15}{36} = \frac{5}{12}\]

Подія \(A \cap B\): перший парний і сума \(> 7\):

Перший кубик: 2 → другий \(> 5\): {6} → 1 спосіб Перший кубик: 4 → другий \(> 3\): {4,5,6} → 3 способи Перший кубик: 6 → другий \(> 1\): {2,3,4,5,6} → 5 способів

\[P(A \cap B) = \frac{1+3+5}{36} = \frac{9}{36} = \frac{1}{4}\]

Перевірка незалежності \(P(A) \cdot P(B) = \frac{1}{2} \cdot \frac{5}{12} = \frac{5}{24} \neq \frac{1}{4} = P(A \cap B)\).

Висновок: події \(A\) і \(B\) є залежними.

Розв’язання в R

expand_grid(d1 = 1:6, d2 = 1:6) |>
  mutate(
    A   = d1 %% 2 == 0,
    B   = (d1 + d2) > 7,
    AnB = A & B
  ) |>
  summarise(
    `P(A)`       = mean(A),
    `P(B)`       = mean(B),
    `P(A∩B)`     = mean(AnB),
    `P(A)×P(B)`  = mean(A) * mean(B),
    `Незалежні`  = abs(mean(AnB) - mean(A) * mean(B)) < 1e-10
  )

Теорема Байєса та повна ймовірність

Задача 5: Контроль якості на заводі

На заводі працюють три верстати, які виробляють відповідно 40%, 35% і 25% загального обсягу продукції. Відсоток бракованих виробів: 5%, 4% і 2% відповідно. Навмання вибрано бракований виріб. Знайти ймовірність того, що він виготовлений на першому верстаті.

Математичне розв’язання

Введемо гіпотези:

Гіпотеза Зміст \(P(H_i)\) \(P(A \mid H_i)\)
\(H_1\) 1-й верстат 0,40 0,05
\(H_2\) 2-й верстат 0,35 0,04
\(H_3\) 3-й верстат 0,25 0,02

Формула повної ймовірності (подія \(A\) — деталь бракована):

\[P(A) = \sum_{i=1}^{3} P(H_i) \cdot P(A \mid H_i)\]

\[P(A) = 0{,}4 \cdot 0{,}05 + 0{,}35 \cdot 0{,}04 + 0{,}25 \cdot 0{,}02 = 0{,}020 + 0{,}014 + 0{,}005 = 0{,}039\]

Формула Байєса:

\[P(H_1 \mid A) = \frac{P(H_1) \cdot P(A \mid H_1)}{P(A)} = \frac{0{,}4 \cdot 0{,}05}{0{,}039} = \frac{0{,}020}{0{,}039} \approx 0{,}513\]

Розв’язання в R

df_bayes5 <- tibble(
  Верстат    = factor(paste("Верстат", 1:3)),
  prior      = c(0.40, 0.35, 0.25),
  likelihood = c(0.05, 0.04, 0.02)
) |>
  mutate(
    joint     = prior * likelihood,
    posterior = joint / sum(joint)
  )

cat(sprintf("Повна ймовірність P(A) = %.4f\n\n", sum(df_bayes5$joint)))
Повна ймовірність P(A) = 0.0390
df_bayes5 |>
  select(Верстат, posterior) |>
  mutate(`P(Hi|A)` = sprintf("%.4f (%.1f%%)", posterior, posterior * 100)) |>
  select(-posterior) |>
  print()
# A tibble: 3 × 2
  Верстат   `P(Hi|A)`     
  <fct>     <chr>         
1 Верстат 1 0.5128 (51.3%)
2 Верстат 2 0.3590 (35.9%)
3 Верстат 3 0.1282 (12.8%)

Задача 6: Біноміальний розподіл

Ймовірність потрапити в ціль при одному пострілі дорівнює \(p = 0{,}7\). Стрілець робить \(n = 8\) пострілів. Знайти:

а) ймовірність рівно 5 влучень; б) ймовірність хоча б одного влучення; в) математичне сподівання та дисперсію числа влучень.

Математичне розв’язання

\(X \sim \mathrm{Bin}(n = 8,\; p = 0{,}7)\)

а) Ймовірність рівно 5 влучень:

\[P(X = 5) = \binom{8}{5} \cdot 0{,}7^5 \cdot 0{,}3^3 = 56 \cdot 0{,}16807 \cdot 0{,}027 \approx 0{,}2541\]

б) Ймовірність хоча б одного влучення:

\[P(X \geq 1) = 1 - P(X = 0) = 1 - \binom{8}{0} \cdot 0{,}7^0 \cdot 0{,}3^8 = 1 - 0{,}3^8 \approx 1 - 0{,}000066 \approx 0{,}9999\]

в) Математичне сподівання та дисперсія:

\[\mathbb{E}[X] = np = 8 \cdot 0{,}7 = 5{,}6\] \[\mathbb{D}[X] = np(1-p) = 8 \cdot 0{,}7 \cdot 0{,}3 = 1{,}68\]

Розв’язання в R

n <- 8; p <- 0.7

# а) P(X = 5)
p_5 <- dbinom(5, size = n, prob = p)
cat(sprintf("а) P(X = 5) = %.4f\n", p_5))
а) P(X = 5) = 0.2541
# б) P(X >= 1) = 1 - P(X = 0)
p_at_least_1 <- 1 - dbinom(0, size = n, prob = p)
cat(sprintf("б) P(X ≥ 1) = %.6f\n", p_at_least_1))
б) P(X ≥ 1) = 0.999934
# в) Математичне сподівання та дисперсія
mean_X <- n * p
var_X  <- n * p * (1 - p)
cat(sprintf("в) E[X] = %.2f, D[X] = %.2f\n", mean_X, var_X))
в) E[X] = 5.60, D[X] = 1.68

Візуалізація розподілу:

  mutate(
    prob      = dbinom(x, size = n, prob = p),
    highlight = x == 5
  ) |>

       x = "Кількість влучень", y = "Ймовірність") +
  theme_minimal(base_size = 12)

Задача 7: Розподіл Пуассона

На телефонну станцію в середньому надходить \(\lambda = 4\) дзвінки на хвилину. Знайти ймовірність того, що за одну хвилину надійде:

а) рівно 6 дзвінків; б) не більше 2 дзвінків; в) більше 5 дзвінків.

Математичне розв’язання

\(X \sim \mathrm{Pois}(\lambda = 4)\), де \(P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}\)

а) Ймовірність рівно 6 дзвінків:

\[P(X = 6) = \frac{4^6 \cdot e^{-4}}{6!} = \frac{4096 \cdot 0{,}01832}{720} \approx 0{,}1042\]

б) Ймовірність не більше 2 дзвінків:

\[P(X \leq 2) = \sum_{k=0}^{2} \frac{4^k e^{-4}}{k!} = e^{-4}\left(1 + 4 + \frac{16}{2}\right) = e^{-4} \cdot 13 \approx 0{,}2381\]

в) Ймовірність більше 5 дзвінків:

\[P(X > 5) = 1 - P(X \leq 5) \approx 1 - 0{,}7851 = 0{,}2149\]

Розв’язання в R

lambda <- 4

# а) P(X = 6)
p_6 <- dpois(6, lambda = lambda)
cat(sprintf("а) P(X = 6) = %.4f\n", p_6))
а) P(X = 6) = 0.1042
# б) P(X <= 2)
p_le2 <- ppois(2, lambda = lambda)
cat(sprintf("б) P(X ≤ 2) = %.4f\n", p_le2))
б) P(X ≤ 2) = 0.2381
# в) P(X > 5)
p_gt5 <- 1 - ppois(5, lambda = lambda)
cat(sprintf("в) P(X > 5) = %.4f\n", p_gt5))
в) P(X > 5) = 0.2149

Візуалізація розподілу Пуассона:

tibble(x = 0:15) |>

    prob = dpois(x, lambda = lambda),

    )
  ) |>
  ggplot(aes(x = factor(x), y = prob, fill = zone)) +
  geom_col(width = 0.7) +
  scale_fill_manual(values = c(
    "6 дзвінків"   = "#d73027",
    "≤ 2 дзвінки"  = "#4393c3",
    "> 5 дзвінків" = "#f4a582",
    "інше"         = "grey70"
  )) +
  labs(title = expression(paste("Pois(", lambda, " = 4)")),
       x = "Кількість дзвінків", y = "Ймовірність", fill = NULL) +
  theme_minimal(base_size = 12) +
  theme(legend.position = "top")

Задача 8 (підвищена складність): Урнова модель та формула Байєса

В урні № 1 — 3 червоних і 5 синіх кульок. В урні № 2 — 6 червоних і 2 синіх. Навмання вибирають одну урну (рівноймовірно), а з неї — одну кульку. Виявилося, що куля червона. Яка ймовірність, що її вийнято з першої урни?

Математичне розв’язання

Гіпотези: \(H_1\) — обрано урну № 1, \(H_2\) — обрано урну № 2.

\[P(H_1) = P(H_2) = \frac{1}{2}\]

Правдоподібності (ймовірність витягнути червону):

\[P(A \mid H_1) = \frac{3}{8}, \quad P(A \mid H_2) = \frac{6}{8} = \frac{3}{4}\]

Повна ймовірність:

\[P(A) = \frac{1}{2} \cdot \frac{3}{8} + \frac{1}{2} \cdot \frac{3}{4} = \frac{3}{16} + \frac{6}{16} = \frac{9}{16}\]

Формула Байєса:

\[P(H_1 \mid A) = \frac{P(H_1) \cdot P(A \mid H_1)}{P(A)} = \frac{\frac{1}{2} \cdot \frac{3}{8}}{\frac{9}{16}} = \frac{\frac{3}{16}}{\frac{9}{16}} = \frac{3}{9} = \frac{1}{3}\]

Розв’язання в R

df_bayes8 <- tibble(
  hypothesis = c("H1", "H2"),
  prior      = c(0.5, 0.5),
  likelihood = c(3/8, 6/8)
) |>
  mutate(
    joint     = prior * likelihood,
    posterior = joint / sum(joint)
  )

cat(sprintf("P(A) = %.4f\n", sum(df_bayes8$joint)))
P(A) = 0.5625
df_bayes8 |>
  select(hypothesis, posterior) |>
  mutate(label = sprintf("%.4f", posterior)) |>
  print()

  hypothesis posterior label 
  <chr>          <dbl> <chr>

2 H2             0.667 0.6667

Задача 9: Діагностичний тест

Діагностичний тест на певне захворювання має чутливість 95% (true positive rate) і специфічність 90% (true negative rate). Поширеність захворювання в популяції становить 1%. Яка ймовірність того, що людина з позитивним результатом тесту дійсно хвора?

Математичне розв’язання

Визначимо події:

  • \(D\) — людина хвора, \(P(D) = 0{,}01\), \(P(\bar{D}) = 0{,}99\)
  • \(T^+\) — тест позитивний

З умови:

\[P(T^+ \mid D) = 0{,}95 \quad \text{(чутливість)}\] \[P(T^- \mid \bar{D}) = 0{,}90 \Rightarrow P(T^+ \mid \bar{D}) = 0{,}10 \quad \text{(1 − специфічність)}\]

Повна ймовірність позитивного тесту:

\[P(T^+) = P(T^+ \mid D)\cdot P(D) + P(T^+ \mid \bar{D})\cdot P(\bar{D})\] \[P(T^+) = 0{,}95 \cdot 0{,}01 + 0{,}10 \cdot 0{,}99 = 0{,}0095 + 0{,}099 = 0{,}1085\]

Формула Байєса:

\[P(D \mid T^+) = \frac{P(T^+ \mid D)\cdot P(D)}{P(T^+)} = \frac{0{,}95 \cdot 0{,}01}{0{,}1085} = \frac{0{,}0095}{0{,}1085} \approx 0{,}0876\]

Парадокс базового рівня: незважаючи на високу точність тесту, лише ≈ 8,8% людей із позитивним результатом справді хворі — через низьку поширеність захворювання.

Задача 10: Дискретна випадкова величина (брокер)

Нехай \(X\) — кількість успішних угод брокера за день, яка приймає значення \(0, 1, 2, 3, 4\). Розподіл ймовірностей:

\[P(0) = 0{,}1, \quad P(1) = 0{,}2, \quad P(2) = 0{,}2, \quad P(3) = 0{,}2\]

  1. Знайти \(P(4)\).
  2. Знайти ймовірність того, що брокер укладе хоча б 3 угоди.
  3. Побудувати функцію розподілу \(F(x)\).
  4. Обчислити \(\mathbb{E}(X)\) та \(\mathrm{Var}(X)\).

Математичне розв’язання

1) Знаходження \(P(4)\):

Оскільки сума всіх ймовірностей дорівнює 1:

\[P(4) = 1 - \sum_{k=0}^{3} P(k) = 1 - (0{,}1 + 0{,}2 + 0{,}2 + 0{,}2) = 1 - 0{,}7 = 0{,}3\]

2) \(P(X \geq 3)\):

\[P(X \geq 3) = P(3) + P(4) = 0{,}2 + 0{,}3 = 0{,}5\]

3) Функція розподілу \(F(x) = P(X \leq x)\):

\[F(x) = \begin{cases} 0 & x < 0 \\ 0{,}1 & 0 \leq x < 1 \\ 0{,}3 & 1 \leq x < 2 \\ 0{,}5 & 2 \leq x < 3 \\ 0{,}7 & 3 \leq x < 4 \\ 1{,}0 & x \geq 4 \end{cases}\]

4) Математичне сподівання та дисперсія:

\[\mathbb{E}(X) = \sum_{k=0}^{4} k \cdot P(k) = 0 \cdot 0{,}1 + 1 \cdot 0{,}2 + 2 \cdot 0{,}2 + 3 \cdot 0{,}2 + 4 \cdot 0{,}3\] \[= 0 + 0{,}2 + 0{,}4 + 0{,}6 + 1{,}2 = 2{,}4\]

\[\mathbb{E}(X^2) = \sum_{k=0}^{4} k^2 \cdot P(k) = 0^2 \cdot 0{,}1 + 1^2 \cdot 0{,}2 + 2^2 \cdot 0{,}2 + 3^2 \cdot 0{,}2 + 4^2 \cdot 0{,}3\] \[= 0 + 0{,}2 + 0{,}8 + 1{,}8 + 4{,}8 = 7{,}6\]

\[\mathrm{Var}(X) = \mathbb{E}(X^2) - [\mathbb{E}(X)]^2 = 7{,}6 - (2{,}4)^2 = 7{,}6 - 5{,}76 = 1{,}84\]

Задача 11: Експоненціальний розподіл (фінтех)

Фінтех-компанія аналізує ефективність підтримки клієнтів. Час, який консультант витрачає на одну складну фінансову справу, має експоненціальний розподіл із середнім значенням 2 години.

  1. Визначте функцію густини \(f(x)\).
  2. Знайдіть ймовірність того, що консультант завершить справу менш ніж за 1 годину.
  3. Компанія хоче запровадити гарантію «Fast Track». Яка ймовірність, що справа займе більше 4 годин?
  4. Знайдіть \(\mathrm{Var}(X)\).

Математичне розв’язання

Середнє \(\mu = 2\) год, отже інтенсивність \(\lambda = \frac{1}{\mu} = \frac{1}{2} = 0{,}5\) год\(^{-1}\).

1) Функція густини:

\[f(x) = \lambda e^{-\lambda x} = 0{,}5\, e^{-0{,}5x}, \quad x > 0; \qquad f(x) = 0, \quad x \leq 0\]

Перевірка нормування:

\[\int_0^{+\infty} 0{,}5\, e^{-0{,}5x}\,dx = 0{,}5 \cdot \left[\frac{e^{-0{,}5x}}{-0{,}5}\right]_0^{+\infty} = \left[-e^{-0{,}5x}\right]_0^{+\infty} = 0 - (-1) = 1\] ✓

2) \(P(X < 1)\):

Знаходимо первісну підінтегральної функції:

\[\int 0{,}5\, e^{-0{,}5x}\, dx = 0{,}5 \cdot \frac{e^{-0{,}5x}}{-0{,}5} = -e^{-0{,}5x} + C\]

Підставляємо межі:

\[P(X < 1) = \int_0^1 0{,}5\, e^{-0{,}5x}\, dx = \left[-e^{-0{,}5x}\right]_0^1 = \left(-e^{-0{,}5 \cdot 1}\right) - \left(-e^{-0{,}5 \cdot 0}\right)\] \[= -e^{-0{,}5} + e^{0} = 1 - e^{-0{,}5} \approx 1 - 0{,}6065 \approx 0{,}3935\]

3) \(P(X > 4)\):

Невласний інтеграл обчислюємо через границю:

\[P(X > 4) = \int_4^{+\infty} 0{,}5\, e^{-0{,}5x}\, dx = \lim_{b \to +\infty} \left[-e^{-0{,}5x}\right]_4^{b}\] \[= \lim_{b \to +\infty}\left(-e^{-0{,}5b} + e^{-0{,}5 \cdot 4}\right) = 0 + e^{-2} \approx 0{,}1353\]

4) Дисперсія:

Спочатку знаходимо \(\mathbb{E}(X)\) і \(\mathbb{E}(X^2)\) методом інтегрування частинами (\(\int u\,dv = uv - \int v\,du\)):

\[\mathbb{E}(X) = \int_0^{+\infty} x \cdot \lambda e^{-\lambda x}\,dx\]

Покладемо \(u = x\), \(dv = \lambda e^{-\lambda x}\,dx\), тоді \(du = dx\), \(v = -e^{-\lambda x}\):

\[\mathbb{E}(X) = \left[-x e^{-\lambda x}\right]_0^{+\infty} + \int_0^{+\infty} e^{-\lambda x}\,dx = 0 + \left[\frac{-e^{-\lambda x}}{\lambda}\right]_0^{+\infty} = \frac{1}{\lambda} = \frac{1}{0{,}5} = 2\]

\[\mathbb{E}(X^2) = \int_0^{+\infty} x^2 \cdot \lambda e^{-\lambda x}\,dx\]

Покладемо \(u = x^2\), \(dv = \lambda e^{-\lambda x}\,dx\), тоді \(du = 2x\,dx\), \(v = -e^{-\lambda x}\):

\[\mathbb{E}(X^2) = \left[-x^2 e^{-\lambda x}\right]_0^{+\infty} + 2\int_0^{+\infty} x e^{-\lambda x}\,dx = 0 + \frac{2}{\lambda} \cdot \mathbb{E}(X) = \frac{2}{\lambda^2}\]

Отже:

\[\mathrm{Var}(X) = \mathbb{E}(X^2) - [\mathbb{E}(X)]^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2} = \frac{1}{0{,}5^2} = 4 \text{ год}^2 \quad \Rightarrow \quad \sigma = 2 \text{ год}\]

Задача 12: Неперервний розподіл (частка ринку)

Антимонопольне агентство вивчає нішу ринку цифрових послуг. Нехай \(X\) — частка ринку випадково вибраної фірми (\(0 \leq X \leq 1\)). Функція густини:

\[f(x) = \begin{cases} 3x^2 & 0 \leq x \leq 1 \\ 0 & \text{інакше} \end{cases}\]

  1. Перевірити, що \(f(x)\) — коректна функція густини. Знайти \(F(x)\).
  2. Обчислити \(\mathbb{E}(X)\) та \(\mathrm{Var}(X)\).
  3. Знайти ймовірність того, що фірма займає домінуюче становище (частка ринку > 80%).

Математичне розв’язання

1) Перевірка та CDF:

Невід’ємність: \(f(x) = 3x^2 \geq 0\) для всіх \(x \in [0,1]\). ✓

Нормування: використовуємо степеневе правило \(\int x^n\,dx = \frac{x^{n+1}}{n+1} + C\):

\[\int_{-\infty}^{+\infty} f(x)\,dx = \int_0^1 3x^2\,dx = 3 \cdot \left[\frac{x^3}{3}\right]_0^1 = \left[x^3\right]_0^1 = 1^3 - 0^3 = 1\] ✓

Функція розподілу \(F(x) = P(X \leq x)\):

При \(0 \leq x \leq 1\) інтегруємо від 0 до \(x\):

\[F(x) = \int_0^x 3t^2\,dt = 3 \cdot \left[\frac{t^3}{3}\right]_0^x = \left[t^3\right]_0^x = x^3 - 0^3 = x^3\]

Повна формула:

\[F(x) = \begin{cases} 0 & x < 0 \\ x^3 & 0 \leq x \leq 1 \\ 1 & x > 1 \end{cases}\]

2) Математичне сподівання та дисперсія:

\[\mathbb{E}(X) = \int_0^1 x \cdot f(x)\,dx = \int_0^1 x \cdot 3x^2\,dx = \int_0^1 3x^3\,dx\] \[= 3 \cdot \left[\frac{x^4}{4}\right]_0^1 = 3 \cdot \frac{1^4 - 0^4}{4} = \frac{3}{4} = 0{,}75\]

\[\mathbb{E}(X^2) = \int_0^1 x^2 \cdot f(x)\,dx = \int_0^1 x^2 \cdot 3x^2\,dx = \int_0^1 3x^4\,dx\] \[= 3 \cdot \left[\frac{x^5}{5}\right]_0^1 = 3 \cdot \frac{1^5 - 0^5}{5} = \frac{3}{5} = 0{,}6\]

Приводимо дроби до спільного знаменника 80:

\[\mathrm{Var}(X) = \mathbb{E}(X^2) - [\mathbb{E}(X)]^2 = \frac{3}{5} - \left(\frac{3}{4}\right)^2 = \frac{3}{5} - \frac{9}{16} = \frac{3 \cdot 16}{80} - \frac{9 \cdot 5}{80} = \frac{48 - 45}{80} = \frac{3}{80} \approx 0{,}0375\]

3) Домінуюче становище:

Використовуємо знайдену \(F(x) = x^3\):

\[P(X > 0{,}8) = 1 - F(0{,}8) = 1 - (0{,}8)^3 = 1 - 0{,}512 = 0{,}488\]

  2. Яка ймовірність, що він реалізує рівно 7 кидків?
  3. Знайдіть математичне сподівання та дисперсію числа реалізованих кидків.

Математичне розв’язання

\(X \sim \mathrm{Bin}(n = 10,\; p = 0{,}8)\), де \(P(X = k) = \binom{10}{k} 0{,}8^k \cdot 0{,}2^{10-k}\).

1) \(P(X \leq 7)\) — через доповнення:

Обчислимо кожен доданок:

\[P(X \leq 7) = 1 - (0{,}3020 + 0{,}2684 + 0{,}1074) = 1 - 0{,}6778 \approx 0{,}3222\]

2) \(P(X = 7)\):

\[P(X = 7) = \binom{10}{7} \cdot 0{,}8^7 \cdot 0{,}2^3 = 120 \cdot 0{,}20972 \cdot 0{,}008 = 120 \cdot 0{,}001678 \approx 0{,}2013\]

Задача 14: Розподіл Пуассона + Біноміальний (кремнієві пластини)

Виробник електроніки випускає кремнієві пластини. Кількість дрібних дефектів на 1 м² матеріалу підпорядковується розподілу Пуассона з \(\lambda = 2\) дефекти/м². Пластина 1 м² вважається «браком», якщо містить більше 3 дефектів; інакше — «якісною».

  • Обчисліть ймовірність \(p\) того, що одна пластина буде бракованою.
  • Партія складається з \(n = 1000\) незалежних пластин; \(Y\) — кількість якісних пластин. Знайдіть ймовірність:
    • усі пластини якісні,
    • хоча б 900 пластин якісні,
    • не більше 100 пластин бракованих.

Математичне розв’язання

Крок 1. Імовірність браку (\(X \sim \mathrm{Pois}(\lambda = 2)\)):

\[p = P(X > 3) = 1 - P(X \leq 3) = 1 - \sum_{k=0}^{3} \frac{2^k e^{-2}}{k!}\]

Розпишемо кожен доданок:

\[P(X=0) = \frac{2^0 e^{-2}}{0!} = e^{-2} \approx 0{,}1353\]

\[P(X=1) = \frac{2^1 e^{-2}}{1!} = 2e^{-2} \approx 0{,}2707\]

\[P(X=2) = \frac{2^2 e^{-2}}{2!} = \frac{4e^{-2}}{2} = 2e^{-2} \approx 0{,}2707\]

\[P(X=3) = \frac{2^3 e^{-2}}{3!} = \frac{8e^{-2}}{6} = \frac{4}{3}e^{-2} \approx 0{,}1804\]

\[P(X \leq 3) = e^{-2}\left(1 + 2 + 2 + \frac{4}{3}\right) = e^{-2} \cdot \frac{19}{3} \approx 0{,}8571\]

\[\boxed{p = 1 - 0{,}8571 = 0{,}1429}, \quad q = 1 - p \approx 0{,}8571\]

Крок 2. Кількість якісних пластин \(Y \sim \mathrm{Bin}(n = 1000,\; q = 0{,}8571)\):

Усі пластини якісні:

\[P(Y = 1000) = q^{1000} = e^{1000 \ln(0{,}8571)} \approx e^{-154{,}2} \approx 0\]

Хоча б 900 якісних — нормальна апроксимація (\(n\) велике):

\[\mu = nq = 1000 \cdot 0{,}8571 = 857{,}1, \quad \sigma = \sqrt{npq} = \sqrt{1000 \cdot 0{,}8571 \cdot 0{,}1429} \approx 11{,}07\]

\[P(Y \geq 900) \approx P\!\left(Z \geq \frac{900 - 857{,}1}{11{,}07}\right) = P(Z \geq 3{,}876) \approx 0{,}0001\]

Не більше 100 бракованих:

Кількість браку \(= 1000 - Y\), тому

\[P(1000 - Y \leq 100) = P(Y \geq 900) \approx 0{,}0001\]

Це та сама подія, що й попередня.

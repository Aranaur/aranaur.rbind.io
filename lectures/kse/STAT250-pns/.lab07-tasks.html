Перевірка узгодженості: ключові ідеї

Що ми вивчаємо сьогодні

Минулого разу ми порівнювали середні (\(t\)-тести). Сьогодні питання інше: чи узгоджуються дані з певним розподілом або чи пов’язані дві ознаки?

Інструмент Питання R
Шапіро–Вілка Чи \(X \sim\) нормальний? shapiro.test
Колмогорова–Смирнова Чи \(F(x) = F_0(x)\)? ks.test
\(\chi^2\) узгодженості Чи частоти \(=\) очікуваним? chisq.test(x, p=)
\(\chi^2\) незалежності Чи дві ознаки незалежні? chisq.test(table)

Перші два перевіряють форму неперервного розподілу. Останні два працюють із частотами категорій.

Розподіли, пов’язані з нормальним

Усі сьогоднішні тести спираються на три «похідні» від нормального розподілу:

\[ \chi^2_n = \sum_{i=1}^{n} Z_i^2, \qquad t_n = \frac{Z}{\sqrt{U/n}}, \qquad F_{d_1,d_2} = \frac{U/d_1}{V/d_2} \]

де \(Z, Z_i \sim \mathcal{N}(0,1)\), а \(U \sim \chi^2_{d_1}\), \(V \sim \chi^2_{d_2}\) — незалежні.

Статистика \(\chi^2\) невід’ємна й праворуч скошена; великі значення → сильна розбіжність.

Загальна логіка

Усі чотири тести мають однакову структуру:

\[ \text{Статистика} \;\xrightarrow{H_0}\; \text{відомий розподіл} \;\Rightarrow\; p\text{-value} \]

Крок Тести нормальності Тести \(\chi^2\)
\(H_0\) дані нормальні частоти \(=\) очікуваним / ознаки незалежні
Статистика \(W\) або \(D_n\) \(\hat{\chi}^2 = \sum \frac{(O_i - E_i)^2}{E_i}\)
Рішення \(p < \alpha \Rightarrow\) відхиляємо \(H_0\) \(p < \alpha \Rightarrow\) відхиляємо \(H_0\)

Для \(\chi^2\)-тестів очікувані частоти \(E_i\) мають бути \(\geq 5\) у кожній категорії.

Перевірка нормальності

Задача 1: Тест Шапіро–Вілка (iris)

Датасет iris — виміри 150 квіток трьох видів. Розглянемо довжину чашолистка (Sepal.Length) лише для виду Setosa (\(n = 50\)).

Чи можна вважати довжину чашолистка Setosa нормально розподіленою? \(\alpha = 0.05\). \[H_0: X \sim \mathcal{N}(\mu, \sigma^2) \quad\text{проти}\quad H_a: X \nsim \mathcal{N}(\mu, \sigma^2)\]


Підказка: статистика Шапіро–Вілка \(W \in (0, 1]\); значення близькі до 1 свідчать на користь нормальності. Малий \(p\)-value → відхиляємо \(H_0\).

Задача 1: Діагностичні графіки

setosa <- iris$Sepal.Length[iris$Species == "setosa"]

p_hist <- tibble(x = setosa) |>
  ggplot(aes(x)) +

                 fill = "#4393c3", color = "white", alpha = 0.8) +
  stat_function(fun = dnorm, args = list(mean = mean(setosa), sd = sd(setosa)),

  theme_minimal(base_size = 12)

  ggplot(aes(sample = x)) +
  stat_qq(color = "#2166ac") +
  stat_qq_line(color = "#d73027", linewidth = 1) +
  labs(title = "Q–Q графік", x = "Теоретичні квантилі", y = "Вибіркові квантилі") +
  theme_minimal(base_size = 12)

p_hist + p_qq

Задача 1: Розв’язання в R

shapiro.test(setosa)

    Shapiro-Wilk normality test

data:  setosa
W = 0.9777, p-value = 0.4595


\(W \approx 0.978\), \(p \approx 0.46 > 0.05\) → не відхиляємо \(H_0\).

Точки на Q–Q графіку лягають близько до діагоналі, гістограма симетрична — припущення про нормальність виправдане. Це означає, що для цих даних коректно застосовувати \(t\)-тести.

Задача 2: Тест Колмогорова–Смирнова (penguins)

Датасет palmerpenguins::penguins — виміри пінгвінів Антарктики. Візьмемо масу тіла (body_mass_g) виду Adelie, прибравши пропуски (\(n = 151\)).

Чи відповідає стандартизована маса тіла Adelie стандартному нормальному розподілу? \[H_0: F(x) = \Phi(x) \quad\text{проти}\quad H_a: F(x) \ne \Phi(x)\]


Підказка: статистика \(D_n = \sup_x |\hat{F}_n(x) - F_0(x)|\) — максимальна відстань між емпіричною та теоретичною функціями розподілу.

Задача 2: Розв’язання в R

adelie <- penguins |>
  filter(species == "Adelie", !is.na(body_mass_g))

z <- as.numeric(scale(adelie$body_mass_g))   # стандартизація: (x - x̄) / s

ks.test(z, "pnorm", mean = 0, sd = 1)

    Asymptotic one-sample Kolmogorov-Smirnov test

data:  z
D = 0.072461, p-value = 0.4061
alternative hypothesis: two-sided

Якщо у даних є повтори (ties), \(p\)-value наближений → R видає попередження. Коли \(\mu, \sigma\) оцінено з тієї ж вибірки, формально точніший — тест Ліллієфорса (nortest::lillie.test).

Задача 2: Порівняння з Шапіро–Вілка

shapiro.test(adelie$body_mass_g)$p.value
[1] 0.03239702

KS: \(D \approx 0.07\), \(p \approx 0.41\) → не відхиляємо \(H_0\). Шапіро–Вілка: \(p \approx 0.032 < 0.05\) → відхиляє \(H_0\). Висновки різні!

Шапіро–Вілка потужніший: при \(n = 151\) він уловлює незначне відхилення від нормальності, якого KS не помічає. KS універсальніший (будь-яке \(F_0\)), але менш чутливий. На практиці для нормальності надають перевагу Шапіро–Вілка, доповнюючи його Q–Q графіком.

Хі-квадрат: тест узгодженості

Задача 3: Чи чесна гральна кістка?

Шестигранну кістку кинули 60 разів і отримали такі частоти граней:

Грань 1 2 3 4 5 6
\(O_i\) 13 8 12 10 7 10

На рівні \(\alpha = 0.05\) перевірте, чи кістка чесна (рівномірний розподіл). \[H_0:\; p_i = \tfrac{1}{6}\;\;\forall i \quad\text{проти}\quad H_a:\; \text{хоча б одне } p_i \ne \tfrac{1}{6}\]

Підказка: \(\hat{\chi}^2 = \sum_{i=1}^{k} \dfrac{(O_i - E_i)^2}{E_i}\), де \(E_i = n p_i\), \(df = k - 1\).

Задача 3: Аналітичне рішення

Очікувана частота (однакова для всіх граней):

\[ E_i = n p_i = 60 \cdot \frac{1}{6} = 10 \]

Статистика:

\[ \hat{\chi}^2 = \frac{(13-10)^2 + (8-10)^2 + (12-10)^2 + (10-10)^2 + (7-10)^2 + (10-10)^2}{10} \]

\[ \hat{\chi}^2 = \frac{9 + 4 + 4 + 0 + 9 + 0}{10} = 2.6, \qquad df = 6 - 1 = 5 \]

Критичне значення: \(\chi^2_{0.95,\,5} \approx 11.07\). Оскільки \(2.6 < 11.07\) → не відхиляємо \(H_0\).

Задача 3: Розв’язання в R

observed <- c(13, 8, 12, 10, 7, 10)
chisq.test(observed, p = rep(1/6, 6))

    Chi-squared test for given probabilities

data:  observed
X-squared = 2.6, df = 5, p-value = 0.7614


\(\hat{\chi}^2 = 2.6\), \(p \approx 0.76 \gg 0.05\) → даних недостатньо, щоб засумніватися в чесності кістки.

Задача 4: Чи однаково поширені огранки? (diamonds)

Датасет diamonds (≈ 54 000 діамантів) містить категорію огранки cut: Fair, Good, Very Good, Premium, Ideal.

Чи трапляються всі п’ять огранок однаково часто? \(\alpha = 0.05\). \[H_0:\; p_i = \tfrac{1}{5}\;\;\forall i \quad\text{проти}\quad H_a:\; \text{принаймні одне } p_i \ne \tfrac{1}{5}\]

table(diamonds$cut)

     Fair      Good Very Good   Premium     Ideal 
     1610      4906     12082     13791     21551

Задача 4: Розв’язання в R

counts <- table(diamonds$cut)
chisq.test(counts, p = rep(1/5, 5))

    Chi-squared test for given probabilities

data:  counts
X-squared = 22745, df = 4, p-value < 2.2e-16


\(\hat{\chi}^2 \approx 22\,745\), \(df = 4\), \(p < 2.2\times10^{-16}\) → рішуче відхиляємо \(H_0\).

На відміну від кістки, тут величезна вибірка робить навіть помірні відхилення статистично значущими. Огранки розподілені далеко не рівномірно: Ideal зустрічається у ~13 разів частіше за Fair.

Задача 5: Стать і вид спорту

Дослідження зв’язку між статтю та вподобанням виду спорту (футбол чи теніс), \(n = 60\):

Футбол Теніс Разом
Чоловіки 20 10 30
Жінки 10 20 30
Разом 30 30 60

Чи пов’язані стать і вподобання спорту? \(\alpha = 0.05\). \[H_0:\; \text{ознаки незалежні} \quad\text{проти}\quad H_a:\; \text{ознаки пов'язані}\]

Підказка: \(E_{il} = \dfrac{O_{i\cdot}\,O_{\cdot l}}{n}\), \(\;df = (k-1)(m-1)\).

Задача 5: Аналітичне рішення

Очікувані частоти (за незалежності всі \(E_{il} = \tfrac{30 \cdot 30}{60} = 15\)):

\[ \hat{\chi}^2 = 2 \cdot \frac{(20 - 15)^2}{15} + 2 \cdot \frac{(10 - 15)^2}{15} = 2 \cdot \frac{25}{15} + 2 \cdot \frac{25}{15} = 6.667 \]

\[ df = (2-1)(2-1) = 1, \qquad \chi^2_{0.95,\,1} = 3.84 \]

Оскільки \(6.667 > 3.84\) → відхиляємо \(H_0\).

Для таблиць \(2\times2\) часто застосовують поправку Єйтса на неперервність — R робить це за замовчуванням.

Задача 5: Розв’язання в R

tbl <- matrix(c(20, 10, 10, 20), nrow = 2, byrow = TRUE,
              dimnames = list(Стать = c("Чоловіки", "Жінки"),
                              Спорт = c("Футбол", "Теніс")))

chisq.test(tbl, correct = FALSE)   # без поправки Єйтса — як у ручному розрахунку

    Pearson's Chi-squared test

data:  tbl
X-squared = 6.6667, df = 1, p-value = 0.009823
chisq.test(tbl)$p.value            # з поправкою Єйтса (за замовчуванням)
[1] 0.02013675

Без поправки: \(\hat{\chi}^2 = 6.67\), \(p \approx 0.010\); з поправкою Єйтса: \(p \approx 0.020\). Обидва \(< 0.05\) → стать і спорт пов’язані.

Задача 6: Куріння і стать (MASS::survey)

Датасет MASS::survey — опитування студентів університету. Перевіримо, чи пов’язані стать (Sex) та звичка курити (Smoke).

Чи залежить інтенсивність куріння від статі? \(\alpha = 0.05\). \[H_0:\; \text{Sex та Smoke незалежні} \quad\text{проти}\quad H_a:\; \text{пов'язані}\]

Головні ідеї

  1. Тести нормальності (Шапіро, KS) перевіряють форму неперервного розподілу; Шапіро — потужніший, KS — універсальніший (будь-яке \(F_0\)).
  2. \(\chi^2\)-тести працюють із частотами: узгодженість (один вимір) чи незалежність (таблиця).
  3. Розмір вибірки вирішує: великий \(n\) (діаманти) робить значущим навіть малий ефект — завжди дивіться на розмір відхилення, а не лише на \(p\)-value.
  4. Умова застосовності: очікувані частоти \(E_i \geq 5\); для \(2\times2\) — поправка Єйтса.
  5. Стандартизовані залишки показують, які категорії найбільше порушують \(H_0\).

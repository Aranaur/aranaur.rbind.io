Частина I. Теорія

Section A. Multiple Choice

Q1. Означення умовної ймовірності

\(P(A \mid B) = ?\)

За означенням умовної ймовірності для \(P(B) > 0\):

\[ P(A \mid B) = \frac{P(A \cap B)}{P(B)}. \]

Правильна відповідь: (B).

Інші варіанти:

    1. \(P(A)\cdot P(B)\) — це \(P(A\cap B)\) лише за незалежності.
    1. переплутані чисельник і знаменник (це формула Баєса для \(P(B \mid A)\)).
    1. \(1 - P(A^c \mid B)\) дорівнює \(P(A \mid B)\), але не є означенням, а наслідком аксіом.

Q2. Дисперсія експоненційного розподілу

\(X \sim \mathrm{Exp}(\lambda = 0.5)\). Знайти \(\operatorname{Var}(X)\).

З довідника: \(\operatorname{Var}(X) = \dfrac{1}{\lambda^2}\). Отже,

\[ \operatorname{Var}(X) = \frac{1}{0.5^{\,2}} = \frac{1}{0.25} = 4. \]

Правильна відповідь: (C) 4.

Q3. Лінійність математичного сподівання

\(E[aX + b] = a\,E[X] + b = a\mu + b\) — це властивість лінійності і виконується завжди.

Правильна відповідь: (C) \(E[aX + b] = a\mu + b\).

Q4. Незалежність подій

Подія \(A\) і \(B\) незалежні \(\iff P(A\cap B) = P(A)P(B)\). Звідси (за умови \(P(B)>0\)):

\[ P(A \mid B) = \frac{P(A\cap B)}{P(B)} = \frac{P(A)P(B)}{P(B)} = P(A). \]

Правильна відповідь: (C) \(P(A\mid B) = P(A)\).

Інші варіанти невірні: (A) і (B) загалом хибні; (D) описує взаємну несумісність, а не незалежність.

Q5. Коефіцієнт кореляції

\(\rho \in [-1, 1]\) і \(|\rho|\) вимірює силу лінійного зв’язку.

Правильна відповідь: (C).

    1. — кореляція \(\neq\) причинність.
    1. — \(\rho = 0\) означає відсутність лінійного зв’язку, але змінні можуть бути залежними (напр. \(Y = X^2\) із симетричним \(X\)).
    1. — \(|\rho| \leq 1\) завжди, незалежно від розподілу.

Section C. Validity Check

C1. Чи є \(f(x) = 6x(1-x)\) валідною PDF на \([0,1]\)?

Умова 1 (невід’ємність). На \([0, 1]\): \(x \geq 0\) і \(1-x \geq 0\), тому \(f(x) \geq 0\). ✓

Умова 2 (інтеграл = 1).

\[ \int_0^1 6x(1-x)\,dx = 6\int_0^1 (x - x^2)\,dx = 6\left[\tfrac{x^2}{2} - \tfrac{x^3}{3}\right]_0^1 = 6\left(\tfrac{1}{2} - \tfrac{1}{3}\right) = 6\cdot \tfrac{1}{6} = 1. \; \checkmark \]

Висновок: \(f(x)\) — валідна PDF (це частковий випадок розподілу \(\mathrm{Beta}(2, 2)\)).

f <- function(x) 6 * x * (1 - x)

1 with absolute error < 1.1e-14

C2. (Бонус) Знайти \(k\) для \(f(x) = k\,e^{-3x}\), \(x \geq 0\)

Шукаємо \(k > 0\):

\[ \int_0^{\infty} k\,e^{-3x}\,dx = k \cdot \left[-\tfrac{1}{3} e^{-3x}\right]_0^{\infty} = k \cdot \tfrac{1}{3} \stackrel{!}{=} 1 \;\Longrightarrow\; \boxed{k = 3}. \]

Перевірка:

  • невід’ємність: \(3 e^{-3x} > 0\) ✓
  • інтеграл: \(\int_0^\infty 3 e^{-3x}\,dx = 1\) ✓

Отже \(f(x) = 3 e^{-3x}\) — це PDF експоненційного розподілу з параметром \(\lambda = 3\): \(E[X] = 1/3\), \(\operatorname{Var}(X) = 1/9\).

(a) pnorm(80, ...) = 0.8413. Бал 80 — це 84-й перцентиль: 84.13% когорти набрали менше. На графіку 80 точно збігається з \(\mu + \sigma\), тож це високий, але не екстремальний результат — приблизно один з шести студентів набрав ще більше.

(b) pnorm(50, ...) = 0.1587. Поріг 50 = \(\mu - \sigma\). Завалили мідтерм \(\approx 15.87\%\) когорти. У термінах \(\sigma\)-стандартизованих одиниць: \(z = (50 - 65)/15 = -1\), отже \(\Phi(-1) = 1 - 0.8413 = 0.1587\).

(c) Інтервал \(\mu \pm \sigma\).

\[ P(50 < X < 80) = \underbrace{\Phi(80)}_{0.8413} - \underbrace{\Phi(50)}_{0.1587} = 0.6826. \]

Це — правило 68–95–99.7 (empirical / 68–95–99.7 rule): у межах \(\pm 1\sigma\) нормальний розподіл містить \(\approx 68\%\) маси.

(d) qnorm(0.90, ...) = 84.22. Стипендію отримують ті, хто набрав не менше 84.22 бала. Оскільки 84.22 > 80 = \(\mu + \sigma\), поріг лежить правіше від \(\mu+\sigma\): 90-й перцентил — це більше, ніж 84.13%, що відповідає \(\mu+\sigma\).

(e) qnorm(0.10, ...) = 45.78. До тьюторінгу запрошують студентів із балом до 45.78. Поріг 45.78 < 50 = \(\mu - \sigma\): 10% припадає на ще лівіший «хвіст», ніж 15.87%, тому 10-й перцентил лежить нижче від \(\mu-\sigma\). Стандартизовано: \(z_{0.10} = -1.282 < -1\).

(f) (Бонус) \(P(45.78 < X < 84.22)\). 45.78 і 84.22 — це 10-й і 90-й перцентилі \(X\). Маса між ними дорівнює \(0.90 - 0.10 = 0.80\). Завдяки симетрії нормального розподілу інтервал симетричний навколо \(\mu = 65\) (відхилення \(\pm 19.22\)).

\[ P(45.78 < X < 84.22) = 0.90 - 0.10 = 0.80. \]

Частина II. Прикладні задачі

Problem 1 — The Pugs Scandal

Дано:

Подія Позначення Значення
Частка справжніх мопсів серед заявок \(P(\text{Pug})\) 0.20
Чутливість тесту (sensitivity) \(P(+\mid\text{Pug})\) 0.85
Хибнопозитивний результат \(P(+\mid\overline{\text{Pug}})\) 0.10 
prior_pug   <- 0.20
sens        <- 0.85          # P(+ | Pug)
fpr         <- 0.10          # P(+ | not Pug)
prior_npug  <- 1 - prior_pug
spec        <- 1 - fpr       # P(- | not Pug)

(a) Закон повної ймовірності — \(P(+)\)

\[ P(+) = P(+\mid\text{Pug})\,P(\text{Pug}) + P(+\mid\overline{\text{Pug}})\,P(\overline{\text{Pug}}) = 0.85 \cdot 0.20 + 0.10 \cdot 0.80 \]

\[ = 0.17 + 0.08 = \boxed{0.25}. \]

p_pos <- sens * prior_pug + fpr * prior_npug
p_pos
[1] 0.25

(b) Теорема Баєса — \(P(\text{Pug}\mid +)\)

\[ P(\text{Pug}\mid +) = \frac{P(+\mid\text{Pug})\,P(\text{Pug})}{P(+)} = \frac{0.85 \cdot 0.20}{0.25} = \frac{0.17}{0.25} = \boxed{0.68}. \]

p_pug_given_pos <- sens * prior_pug / p_pos
p_pug_given_pos
[1] 0.68

(c) \(P(\overline{\text{Pug}}\mid -)\)

Спершу — компоненти:

  • \(P(-\mid\overline{\text{Pug}}) = 1 - 0.10 = 0.90\)
  • \(P(-) = 1 - P(+) = 1 - 0.25 = 0.75\)

Тоді:

\[ P(\overline{\text{Pug}}\mid -) = \frac{P(-\mid\overline{\text{Pug}})\,P(\overline{\text{Pug}})}{P(-)} = \frac{0.90 \cdot 0.80}{0.75} = \frac{0.72}{0.75} = \boxed{0.96}. \]

p_neg <- 1 - p_pos
p_npug_given_neg <- spec * prior_npug / p_neg
p_npug_given_neg
[1] 0.96

(d) Інтерпретація 85% «точності»

Заява організатора плутає умовну ймовірність у двох напрямках:

\[ P(+\mid\text{Pug}) = 0.85 \quad\text{але}\quad P(\text{Pug}\mid +) = 0.68. \]

Це base-rate fallacy (помилка ігнорування базової частоти): висока чутливість тесту ще не означає високу post-test ймовірність, якщо апріорна частка \(\text{Pug}\) мала. Уявімо 100 заявок:

  • 20 справжніх мопсів → з них \(0.85 \cdot 20 = 17\) дають «+»
  • 80 не-мопсів → з них \(0.10 \cdot 80 = 8\) дають «+»
  • Усього «+»: 25, з яких 17 справжніх, тобто \(17/25 = 0.68\).

Хибнопозитивні (8) розбавляють пул позитивних, бо їх «постачає» велика популяція не-мопсів.

Problem 2 — Loto-Zabava

Один квиток коштує 10 UAH. Нехай \(X_i\) — чистий виграш по \(i\)-му квитку:

\[ X_i = \begin{cases} +80, & \text{з імовірністю } 0.10 \\ -10, & \text{з імовірністю } 0.90 \end{cases} \]

(a) Очікування і дисперсія однієї ставки

\[ E[X_i] = 80 \cdot 0.10 + (-10) \cdot 0.90 = 8 - 9 = -1 \text{ UAH}. \]

\[ E[X_i^2] = 80^2 \cdot 0.10 + (-10)^2 \cdot 0.90 = 640 + 90 = 730. \]

\[ \operatorname{Var}(X_i) = E[X_i^2] - (E[X_i])^2 = 730 - 1 = 729, \qquad \operatorname{SD}(X_i) = 27. \]

Висновок: середній збиток на квиток дорівнює 1 UAH, тож купівля одного квитка не є вигідним рішенням у середньому.

x   <- c(80, -10)
p   <- c(0.10, 0.90)
EX  <- sum(x * p)
EX2 <- sum(x^2 * p)
VarX <- EX2 - EX^2
SDX  <- sqrt(VarX)
c(EX = EX, EX2 = EX2, Var = VarX, SD = SDX)
 EX EX2 Var  SD 
 -1 730 729  27

(b) Сума за 100 квитків

\(S_{100} = \sum_{i=1}^{100} X_i\). Оскільки \(X_i\) — i.i.d.:

\[ E[S_{100}] = n\mu = 100 \cdot (-1) = -100, \]

\[ \operatorname{Var}(S_{100}) = n\sigma^2 = 100 \cdot 729 = 72\,900, \qquad \operatorname{SD}(S_{100}) = \sqrt{72\,900} = 270. \]

Теорема: Центральна гранична теорема (CLT) — для i.i.d. величин зі скінченними \(\mu\) і \(\sigma^2 > 0\) стандартизована сума прямує до \(\mathcal{N}(0,1)\).

Наближений розподіл:

\[ \boxed{\,S_{100} \;\overset{\text{прибл.}}{\sim}\; \mathcal{N}(-100,\; 270^2)\,}. \]

n  <- 100
mu_S    <- n * EX
var_S   <- n * VarX
sd_S    <- sqrt(var_S)
c(E = mu_S, Var = var_S, SD = sd_S)
    E   Var    SD 
 -100 72900   270

(c) Імовірність вийти «в плюс» — \(P(S_{100} > 0)\)

Стандартизуємо:

\[ P(S_{100} > 0) = P\!\left(\frac{S_{100} - (-100)}{270} > \frac{0 - (-100)}{270}\right) = P\!\left(Z > \frac{100}{270}\right) \]

\[ = P(Z > 0.370) = 1 - \Phi(0.37) \approx 1 - 0.6443 = \boxed{0.3557}. \]

Тобто річний шанс «вийти в нуль або краще» — приблизно 35.6%.

p_profit_100 <- 1 - pnorm(0, mean = mu_S, sd = sd_S)

(d) Закон великих чисел і «прорив» гравця

Формулювання LLN. Для i.i.d. \(X_1, X_2, \ldots\) зі скінченим \(\mu = E[X_i]\),

\[ \bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \;\xrightarrow[n\to\infty]{\text{a.s. / в ім.}}\; \mu = -1. \]

Пояснення. Гравець плутає LLN з «правом на реванш». LLN гарантує, що середній чистий виграш на квиток стабілізується навколо \(-1\) UAH — а не навколо нуля чи додатного числа. Тобто чим більше квитків, тим впевненіший збиток, а не виграш: \(E[S_n] = -n\) — лінійно зростає вниз. Жодне «дострибатися до плюсу» математично не передбачено; залишається лише дисперсія \(\sigma\sqrt{n}\), яка росте повільніше, ніж \(|E[S_n]|\).

set.seed(7)
draws <- sample(x, 5000, replace = TRUE, prob = p)
df_lln <- tibble(
  n = seq_along(draws),
  cum_mean = cumsum(draws) / seq_along(draws)
)

(e) (Бонус) Мінімальне \(n\), при якому \(P(S_n > 0) \leq 0.25\)

Через CLT (для \(E[X_i] = -1\), \(\operatorname{Var}(X_i) = 729\), \(\sigma = 27\)):

\[ P(S_n > 0) = P\!\left(Z > \frac{0 - (-n)}{27\sqrt{n}}\right) = P\!\left(Z > \frac{\sqrt{n}}{27}\right). \]

Умова \(P(S_n > 0) \leq 0.25\) еквівалентна:

\[ \frac{\sqrt{n}}{27} \geq z_{0.75} \approx 0.674 \;\;\Longrightarrow\;\; \sqrt{n} \geq 27 \cdot 0.674 = 18.198 \]

\[ n \geq 18.198^2 \approx 331.16 \;\Longrightarrow\; \boxed{n_{\min} = 332}. \]

z75    <- 0.674
n_min  <- ceiling((27 * z75)^2)

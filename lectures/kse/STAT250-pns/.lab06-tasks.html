Статистичний висновок: ключові ідеї

Що ми вивчаємо сьогодні

Економісти рідко спостерігають усю сукупність → беруть вибірку й роблять висновок.

\[ \text{Вибірка} \;\longrightarrow\; \text{Висновок про сукупність} \]

Інструмент Питання R
Довірчий інтервал Який діапазон для \(\mu\)? t.test, qnorm
Одновибірковий t-тест Чи \(\mu \ne \mu_0\)? t.test(x, mu=)
Двовибірковий / Welch Чи \(\mu_1 \ne \mu_2\)? t.test(x, y)
Парний t-тест Чи \(\mu_D \ne 0\)? t.test(paired=TRUE)
Тест пропорцій Чи \(p_1 \ne p_2\)? prop.test

Дві базові формули

Довірчий інтервал (загальна форма):

\[ \text{Оцінка} \pm \underbrace{\text{Критичне значення}}_{z \text{ або } t} \times \underbrace{\text{Стандартна похибка}}_{\sigma/\sqrt{n}\;\text{або}\;s/\sqrt{n}} \]

Статистика t-тесту:

\[ t = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \sim t_{n-1} \]

Відома \(\sigma\) → \(z\)-розподіл. Невідома \(\sigma\) → \(t\)-розподіл (важчі хвости). Правило рішення: якщо \(p < \alpha\) → відхиляємо \(H_0\).

Довірчі інтервали

Задача 1: Середня зарплата (відома σ)

Аналітик оцінює середню зарплату в IT. Відомо \(\sigma = 18\,000\) грн. Вибірка \(n = 100\) дала \(\bar{X} = 62\,000\) грн.

Побудуйте 95% довірчий інтервал для справжньої середньої зарплати.


Підказка: оскільки \(\sigma\) відома → \(z\)-інтервал:

\[\bar{X} \pm z_{1-\alpha/2}\,\frac{\sigma}{\sqrt{n}}, \qquad z_{0.975} = 1.96\]

Задача 1: Аналітичне рішення

Стандартна похибка:

\[ \text{SE} = \frac{\sigma}{\sqrt{n}} = \frac{18000}{\sqrt{100}} = 1800 \text{ грн} \]

Похибка (margin of error):

\[ \text{ME} = z_{0.975}\cdot\text{SE} = 1.96 \times 1800 = 3528 \text{ грн} \]

Інтервал:

\[ 62000 \pm 3528 = (58\,472;\; 65\,528) \text{ грн} \]

Висновок: з рівнем довіри 95% середня зарплата лежить у межах 58 472 – 65 528 грн.

Задача 1: Розв’язання в R

sigma <- 18000; n <- 100; x_bar <- 62000; alpha <- 0.05

se <- sigma / sqrt(n)
z  <- qnorm(1 - alpha / 2)
me <- z * se
ci <- c(x_bar - me, x_bar + me)

cat(sprintf("SE = %.1f грн\n", se))
SE = 1800.0 грн
cat(sprintf("z(0.975) = %.4f, ME = %.1f грн\n", z, me))
z(0.975) = 1.9600, ME = 3527.9 грн
cat(sprintf("95%% CI: (%.0f; %.0f) грн\n", ci[1], ci[2]))
95% CI: (58472; 65528) грн

Задача 2: Час відгуку API (невідома σ)

Стартап виміряв час відгуку API (мс) у 12 запитах:

118, 124, 131, 109, 127, 140, 115, 122, 133, 119, 126, 130

Побудуйте 97% довірчий інтервал для середнього часу відгуку. Дисперсія сукупності невідома.


Підказка: мала вибірка + невідома \(\sigma\) → \(t\)-розподіл з \(n-1 = 11\) df:

\[\bar{X} \pm t_{1-\alpha/2,\,n-1}\cdot\frac{s}{\sqrt{n}}\]

Задача 2: Розв’язання в R

x <- c(118, 124, 131, 109, 127, 140, 115, 122, 133, 119, 126, 130)
n <- length(x); alpha <- 0.03

x_bar <- mean(x); s <- sd(x); se <- s / sqrt(n)
t_crit <- qt(1 - alpha / 2, df = n - 1)
ci <- c(x_bar - t_crit * se, x_bar + t_crit * se)

cat(sprintf("n = %d, X̄ = %.2f мс, s = %.2f мс\n", n, x_bar, s))
n = 12, X̄ = 124.50 мс, s = 8.54 мс
cat(sprintf("t(0.985, %d) = %.4f\n", n - 1, t_crit))
t(0.985, 11) = 2.4907
cat(sprintf("97%% CI: (%.2f; %.2f) мс\n", ci[1], ci[2]))
97% CI: (118.36; 130.64) мс

Перевірка через t.test():

t.test(x, conf.level = 0.97)$conf.int
[1] 118.3569 130.6431
attr(,"conf.level")
[1] 0.97

Задача 2: Чому інтервал ширший?

Критичне значення \(t_{0.985,\,11} \approx 2.49\) більше за \(z_{0.985} \approx 2.17\).

Задача 3: Одновибірковий t-тест (sleep)

Датасет sleep — приріст годин сну у 10 пацієнтів для двох препаратів. Беремо Препарат 1.

Чи збільшує Препарат 1 тривалість сну більш ніж на 0 годин? \[H_0: \mu = 0 \quad\text{проти}\quad H_a: \mu > 0\]


Підказка: \(t = \dfrac{\bar{X} - \mu_0}{s/\sqrt{n}}\), одностороння альтернатива → відхиляємо при \(t > t_{1-\alpha,\,n-1}\).

Задача 3: Розв’язання в R

data(sleep)
drug1 <- sleep$extra[sleep$group == 1]
drug1
 [1]  0.7 -1.6 -0.2 -1.2 -0.1  3.4  3.7  0.8  0.0  2.0
n <- length(drug1); x_bar <- mean(drug1); s <- sd(drug1)
t_stat <- (x_bar - 0) / (s / sqrt(n))
p_val  <- 1 - pt(t_stat, df = n - 1)   # one-sided (greater)
cat(sprintf("X̄ = %.3f, s = %.3f, t = %.3f, p = %.4f\n", x_bar, s, t_stat, p_val))
X̄ = 0.750, s = 1.789, t = 1.326, p = 0.1088
t.test(drug1, mu = 0, alternative = "greater")

Задача 3: Висновок

\(p \approx 0.11 > 0.05\) → не відхиляємо \(H_0\).

Хоча в середньому Препарат 1 трохи збільшує сон (\(\bar{X} \approx 0.75\) год), доказів статистично значущого ефекту при \(\alpha = 0.05\) недостатньо.

Відсутність значущості \(\ne\) доказ відсутності ефекту — можливо, замала вибірка (низька потужність).

Задача 4: Парний t-тест (sleep)

Кожен пацієнт отримав обидва препарати → спостереження спарені. Нехай \(D = \text{Препарат 2} - \text{Препарат 1}\).

Чи є значуща різниця в ефективності? \[H_0: \mu_D = 0 \quad\text{проти}\quad H_a: \mu_D \ne 0\] Побудуйте 95% довірчий інтервал.


Підказка: парний тест = одновибірковий на різницях \(D_i\): \(\;t = \dfrac{\bar{D}}{s_D/\sqrt{n}}\).

Задача 4: Розв’язання в R

drug2 <- sleep$extra[sleep$group == 2]
d <- drug2 - drug1

n <- length(d); d_bar <- mean(d); s_d <- sd(d)
t_stat <- d_bar / (s_d / sqrt(n))
p_val  <- 2 * (1 - pt(abs(t_stat), df = n - 1))
cat(sprintf("D̄ = %.3f, s_D = %.3f, t = %.3f, p = %.5f\n", d_bar, s_d, t_stat, p_val))
D̄ = 1.580, s_D = 1.230, t = 4.062, p = 0.00283
t.test(drug2, drug1, paired = TRUE, conf.level = 0.95)

Тест пропорцій (A/B)

Задача 6: A/B-тест конверсії

Інтернет-магазин тестує новий дизайн сторінки:

  • Контроль (стара): \(n_1 = 4\,800\), конверсій \(x_1 = 528\)
  • Тест (нова): \(n_2 = 4\,750\), конверсій \(x_2 = 593\)

Чи відрізняються конверсії? \(\alpha = 0.05\). \[H_0: p_{control} = p_{treatment} \quad\text{проти}\quad H_a: p_{control} \ne p_{treatment}\]


Підказка: \(\hat{p} = \dfrac{x_1+x_2}{n_1+n_2}, \quad Z = \dfrac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})\left(\frac{1}{n_1}+\frac{1}{n_2}\right)}}\)

Задача 6: Аналітичне рішення

Вибіркові частки:

\[ \hat{p}_1 = \frac{528}{4800} = 0.1100, \qquad \hat{p}_2 = \frac{593}{4750} = 0.1248 \]

Об’єднана оцінка під \(H_0\):

\[ \hat{p} = \frac{528 + 593}{4800 + 4750} = \frac{1121}{9550} \approx 0.1174 \]

Статистика:

\[ Z = \frac{0.1100 - 0.1248}{\sqrt{0.1174 \cdot 0.8826 \left(\frac{1}{4800}+\frac{1}{4750}\right)}} \approx -2.37 \]

Задача 6: Розв’язання в R

successes <- c(control = 528, treatment = 593)
trials    <- c(control = 4800, treatment = 4750)

prop.test(successes, trials, alternative = "two.sided", correct = FALSE)

Задача 6: Ручна перевірка Z

p1 <- successes[1] / trials[1]
p2 <- successes[2] / trials[2]
p_pool <- sum(successes) / sum(trials)

z <- (p1 - p2) / sqrt(p_pool * (1 - p_pool) * (1/trials[1] + 1/trials[2]))
p_val <- 2 * (1 - pnorm(abs(z)))
cat(sprintf("p̂1 = %.4f, p̂2 = %.4f, p̂_pool = %.4f\n", p1, p2, p_pool))
p̂1 = 0.1100, p̂2 = 0.1248, p̂_pool = 0.1174
cat(sprintf("Z = %.3f, Z² = %.3f, p-value = %.4f\n", z, z^2, p_val))
Z = -2.253, Z² = 5.076, p-value = 0.0243

\(p \approx 0.018 < 0.05\) → відхиляємо \(H_0\): нова сторінка дає вищу конверсію.

Головні ідеї

  1. Невідома \(\sigma\) → \(t\)-розподіл з важчими хвостами; при великому \(n\) зливається з нормальним.
  2. Правильна інтерпретація CI — про процедуру, а не про конкретний інтервал.
  3. Вибір тесту залежить від структури даних: одна вибірка / дві незалежні / спарені / частки.
  4. Welch — безпечне замовчування для двох незалежних вибірок.
  5. Рішення залежить від \(\alpha\) (задача 5): той самий \(p\)-value → різні висновки.
